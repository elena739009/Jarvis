## 1. Цель и аудитория

**Цель сайта:** Личный бренд-лендинг Елены — специалиста по созданию сайтов, визиток и AI-агентов. Одно целевое действие: получить заявку через форму или Telegram.

**Целевая аудитория:**
- Эксперты (коучи, психологи, нутрициологи, консультанты) — нужна цифровая визитка или лендинг
- Владельцы малого бизнеса — нужен сайт или автоматизация
- Стартапы на ранней стадии — нужен быстрый MVP или AI-интеграция

**Ключевое отличие от конкурентов:** Тёплый, человечный подход + честность (учебные проекты обозначены как демо) + игривый личный бренд «Алё...на!»

## 4. Визуальная система

### Цветовая палитра

```css
:root {
  --bg:           #FFF8F3;  /* кремовый фон страницы */
  --surface:      #FFFFFF;  /* фон карточек */
  --text:         #1A1A2E;  /* основной текст */
  --text-muted:   #6B7280;  /* вспомогательный текст */
  --accent:       #FF6B6B;  /* коралловый — кнопки, CTA */
  --accent-2:     #A78BFA;  /* лавандовый — второй акцент */
  --accent-3:     #34D399;  /* мятный — иконки гарантий */
  --border:       #F0E6DF;  /* разделители и границы */
  --shadow:       rgba(255, 107, 107, 0.15); /* тень от карточек */
}
```

### Шрифты

```css
/* Заголовки */
font-family: 'Bricolage Grotesque', sans-serif;
font-weight: 700 / 800;

/* Текст */
font-family: 'Inter', sans-serif;
font-weight: 400 / 500;

/* Адаптивный размер */
--font-hero: clamp(2.5rem, 6vw, 4.5rem);
--font-h2:   clamp(1.8rem, 4vw, 2.8rem);
--font-body: 1rem;
line-height: 1.6;
```

### Анимации

| Элемент | Анимация | Параметры |
|---------|----------|-----------|
| Секции при скролле | fade-in + translateY(30px → 0) | 0.6s ease, CSS view() |
| SVG-аватар | float (покачивание) | 3s ease-in-out infinite |
| Hover на карточках | scale(1.02) + box-shadow | 0.3s ease |
| CTA-кнопка | pulse glow | 2s ease-in-out infinite |
| Мобильная sticky кнопка | slide-up при загрузке | 0.4s ease |

**Правило доступности:** `@media (prefers-reduced-motion: reduce)` — все анимации отключаются.

## 6. UX-стандарты и чек-лист

### Обязательно реализовать:
- [ ] HTTPS (при деплое)
- [ ] `font-display: swap` для Google Fonts
- [ ] `defer` для JS
- [ ] Все изображения с `alt`-текстами
- [ ] `:focus-visible` стили
- [ ] `prefers-reduced-motion: reduce`
- [ ] Семантическая разметка: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
- [ ] Copyright с текущим годом (2026)
- [ ] Форма показывает success/error состояние
- [ ] Тач-цели ≥ 44px на мобильном
- [ ] Мета-теги: title, description, og:image

### Что НЕ делать (из research.md):
- Нет автоплея звука/видео
- Нет полноэкранного прелоадера
- Нет pop-up при входе
- Нет scroll-hijack
- Нет бесконечных анимаций без паузы
- Нет фейковых таймеров
- Нет технического жаргона в текстах

## 8. JS-функциональность (vanilla, без библиотек)

| Функция | Реализация |
|---------|-----------|
| Sticky nav + активный пункт | IntersectionObserver |
| Reveal-анимации при скролле | IntersectionObserver + CSS класс `.visible` |
| Гамбургер-меню | toggle класса `.open` |
| Мобильная sticky CTA | scroll event, показать после 300px |
| Кнопка «наверх» | scroll event, показать после 300px |
| Форма (успех/ошибка) | submit event, показ состояний |
| Плавный скролл к якорям | `scrollIntoView({ behavior: 'smooth' })` |
